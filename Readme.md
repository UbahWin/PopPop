# PopPop
Кроссплатформенная игра, расчитанная в первую очередь для пользователей смартфона, в которой надо Лопать шарики, падающие с неба



## Технологии
- [Unity3D](https://unity.com/ru)
- [C#](https://learn.microsoft.com/ru-ru/dotnet/csharp/)
